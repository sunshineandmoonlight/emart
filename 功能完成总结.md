## 🚀 完整购物流程

### 方式一：普通购物流程
1. 访问主页（无需登录）
2. 浏览商品或搜索商品（无需登录）
3. 点击商品查看详情（无需登录）
4. 添加商品到购物车（需登录）
5. 进入购物车页面（需登录）
6. 点击"结算"按钮
7. 填写收货地址信息
8. 提交订单
9. 查看订单列表

### 方式二：立即购买流程
1. 访问商品详情页（无需登录）
2. 选择数量
3. 点击"立即购买"按钮（需登录）
4. 填写收货地址
5. 提交订单

## 🎯 核心功能亮点

### 1. 类似淘宝的购物体验
- **无需登录浏览**: 主页、商品列表、商品详情页都可以浏览
- **登录拦截**: 购物车、订单、个人中心需要登录
- **搜索功能**: 顶部搜索框，支持回车搜索
- **购物车徽章**: 实时显示购物车商品数量

### 2. 优雅的UI设计
- **渐变色导航栏**: 橙红色渐变（#ff6b6b → #ff8e53）
- **卡片悬停效果**: 鼠标悬停卡片上浮效果
- **响应式布局**: 自适应手机、平板、电脑
- **底部固定栏**: 订单确认页面底部固定提交栏

### 3. 完整的购物流程
- **浏览商品** → **查看详情** → **加入购物车** → **结算** → **填写地址** → **提交订单**
- **立即购买**: 从详情页直接购买，跳过购物车
- **地址管理**: 支持多个地址，可设置默认地址

### 4. 订单管理
- **订单列表**: 查看所有历史订单
- **状态显示**: 订单状态用不同颜色标签显示
- **取消订单**: 待付款订单可以取消
- **订单详情**: 查看订单商品和金额

## 🔧 技术特点

### 后端技术
1. **Spring Boot 2.7.18** - 稳定的企业级框架
2. **MyBatis-Plus** - 简化数据库操作
3. **Spring Security + JWT** - 安全认证
4. **Redis** - 购物车缓存，提升性能
5. **Druid** - 阿里巴巴数据库连接池

### 前端技术
1. **Vue 3** - 最新Vue框架
2. **Vite** - 快速构建工具
3. **Element Plus** - UI组件库
4. **Vue Router 4** - 路由管理
5. **Axios** - HTTP请求

### 设计模式
- **前后端分离**: RESTful API
- **分层架构**: Controller → Service → Mapper
- **缓存优化**: Redis缓存购物车数据
- **数据持久化**: localStorage存储地址信息
