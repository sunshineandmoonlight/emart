# Emart 电商平台 - 完整项目规划

## 📋 项目概述

**项目名称**: Emart 电子商务平台
**项目类型**: B2C 电商平台（类似淘宝、京东）
**技术栈**:
- 后端: Spring Boot 2.7.18 + MyBatis-Plus + MySQL + Redis
- 前端: Vue 3 + Vite + Element Plus
- 认证: JWT + Spring Security
