### 3. 后端API ✅

#### 核心模块

**用户模块 (UMS)**
```java
POST /user/register      // 用户注册
POST /user/login         // 用户登录
POST /user/logout        // 用户注销
GET  /user/list          // 获取用户列表（管理员）
POST /user/status/{id}   // 更新用户状态（管理员）
```

**商品模块 (PMS)**
```java
POST /product/list       // 商品分页查询
POST /product/create     // 创建商品
POST /product/update/{id}// 更新商品
POST /product/delete/{id}// 删除商品
GET  /product/{id}       // 商品详情
```

**订单模块 (OMS)**
```java
POST /order/create       // 创建订单
GET  /order/list         // 用户订单列表
GET  /order/all          // 所有订单（管理员）
POST /order/ship/{id}    // 订单发货（管理员）
GET  /order/stats        // 销售统计（管理员）
POST /order/cancel/{id}  // 取消订单
```

**购物车模块 (OMS)**
```java
POST /cart/add           // 添加到购物车
GET  /cart/list          // 购物车列表
POST /cart/update        // 更新数量
POST /cart/delete        // 删除商品
```

**浏览日志 (CMS)**
```java
POST /browse/log/{id}    // 记录浏览
GET  /browse/logs        // 浏览日志（管理员）
GET  /browse/hot         // 热门商品（管理员）
```

**管理员模块 (Admin)**
```java
POST /admin/login        // 管理员登录
```

#### 技术特点
- RESTful API设计
- JWT Token认证
- MyBatis-Plus数据访问
- Redis缓存购物车
- Druid连接池
- BCrypt密码加密
- CORS跨域支持

## 🚀 项目启动指南

### 环境要求
- JDK 17+
- Node.js 16+
- MySQL 8.0+
- Redis 6.0+

### 启动步骤

#### 1. 启动后端
```bash
cd D:\test\project\emart
./mvnw spring-boot:run
```
访问：http://localhost:8080

#### 2. 启动用户前台
```bash
cd D:\test\project\emart\emart-web
npm run dev
```
访问：http://localhost:5174

#### 3. 启动管理后台
```bash
cd D:\test\project\emart\emart-admin
npm run dev
```
访问：http://localhost:5175
