## 🐛 常见问题排查

### 问题 1: 图片不显示
**检查项：**
- [ ] 确认 uploads 和 images 目录有图片文件
- [ ] 访问 http://localhost:8080/uploads/java-book.jpg 测试图片是否能访问
- [ ] 检查浏览器控制台是否有 404 错误

### 问题 2: 购物车添加失败
**检查项：**
- [ ] 确认 Redis 是否启动：`redis-cli ping` 应返回 PONG
- [ ] 检查后端日志是否有 Redis 连接错误
- [ ] 确认是否已登录（localStorage 中有 token）

### 问题 3: 订单创建失败
**检查项：**
- [ ] 确认购物车有商品
- [ ] 检查收货信息是否完整
- [ ] 查看后端日志中的错误信息
