# 🎨 商品详情页优化指南

## ✅ 当前状态

商品详情页已经具备：
- 基本的图片展示
- 缩略图切换
- 商品信息展示
- 评价系统
- 收藏功能
- 加入购物车

## 🚀 推荐优化项（按优先级）

### 1. 图片展示优化 ⭐⭐⭐

**当前实现：**
- 基础图片展示 + 缩略图切换

**建议优化：**
- 添加图片放大镜效果
- 支持多图轮播
- 添加图片预览（点击放大）

**实现方式：**
```vue
<!-- 放大镜效果 -->
<div class="main-image" @mouseenter="showMagnifier = true">
  <img :src="currentImage" ref="mainImage">
  <div class="magnifier" v-show="showMagnifier">
    <img :src="currentImage" style="transform: scale(2)">
  </div>
</div>
```

### 2. SKU选择功能 ⭐⭐⭐

**当前状态：**
- 商品只有单一价格和库存
- 无法选择不同规格（颜色、尺寸等）

**数据库表已创建：** `03_sku_stock.sql`

**需要实现：**
- 后端：SkuStock实体、Mapper、Service、Controller
- 前端：SKU选择组件
  - 颜色选择器
  - 尺寸选择器
  - 动态价格更新
  - 库存实时显示

**SKU数据结构示例：**
```json
{
  "productId": 1,
  "skus": [
    {
      "id": 1,
      "skuCode": "IP15PRO-256GB-BLACK",
      "spData": "{\"颜色\": \"黑色\", \"容量\": \"256GB\"}",
      "stock": 50,
      "price": 7999.00,
      "image": "/images/iphone15-black.jpg"
    },
    {
      "id": 2,
      "skuCode": "IP15PRO-256GB-WHITE",
      "spData": "{\"颜色\": \"白色\", \"容量\": \"256GB\"}",
      "stock": 30,
      "price": 7999.00,
      "image": "/images/iphone15-white.jpg"
    }
  ]
}
```
