# 🚀 Emart 电商平台 - 快速使用指南

---

## 📋 项目概览

**Emart** 是一个功能完整的B2C电子商务平台，包含用户前台和管理后台两个独立系统。

### 核心功能
- ✅ 用户注册、登录、购物
- ✅ 商品浏览、搜索、分类
- ✅ 购物车管理（Redis缓存）
- ✅ 订单创建、查询、取消
- ✅ 商品管理（增删改查）
- ✅ 订单管理、发货处理
- ✅ 用户管理、销售统计
- ✅ 浏览日志分析

### 技术栈
- **后端**: Spring Boot 2.7.18 + MyBatis-Plus + MySQL + Redis
- **前端**: Vue 3 + Element Plus + Vite
- **认证**: JWT Token
- **缓存**: Redis（购物车）

## 📖 使用指南

### 一、用户端使用

#### 1. 注册账号
1. 访问 http://localhost:5174
2. 点击右上角"注册"
3. 填写用户名、密码、邮箱
4. 点击"注册"按钮

#### 2. 浏览商品
1. **主页**: 查看轮播图、分类、热门商品
2. **搜索**: 顶部搜索框输入关键词
3. **分类**: 点击分类标签筛选商品
4. **详情**: 点击商品卡片查看详情

#### 3. 购物流程
1. **加入购物车**:
   - 在商品详情页选择数量
   - 点击"加入购物车"
   - 查看购物车徽章数量

2. **查看购物车**:
   - 点击顶部购物车图标
   - 修改商品数量
   - 删除不需要的商品

3. **结算订单**:
   - 从购物车点击"结算"
   - 或在商品详情页点击"立即购买"
   - 填写收货地址信息
   - 确认订单信息
   - 点击"提交订单"

4. **查看订单**:
   - 点击顶部用户名
   - 选择"我的订单"
   - 查看订单状态、取消订单

#### 4. 个人中心
- **个人信息**: 查看和编辑基本信息
- **收货地址**: 管理多个收货地址
- **修改密码**: 更新登录密码

---

### 二、管理后台使用

#### 1. 登录管理后台
1. 访问 http://localhost:5175
2. 输入管理员账号: `admin`
3. 输入密码: `admin123`
4. 点击"登录"按钮

#### 2. 首页统计（Dashboard）
登录后自动进入首页，可查看：
- 今日订单数
- 今日销售额
- 总商品数
- 总用户数

#### 3. 商品管理
**商品列表**:
- 点击左侧菜单"商品管理"
- 查看所有商品
- 搜索商品
- 点击"编辑"修改商品
- 点击"删除"删除商品

**添加商品**:
- 点击"添加商品"按钮
- 填写商品信息：
  - 商品名称
  - 副标题
  - 价格
  - 库存
  - 分类
  - 品牌
  - 商品详情
- 点击"提交"保存

**编辑商品**:
- 在商品列表点击"编辑"
- 修改商品信息
- 点击"提交"保存

#### 4. 订单管理
**查看订单**:
- 点击左侧菜单"订单管理"
- 查看所有订单
- 查看订单状态：
  - 待付款（黄色）
  - 已付款（绿色）
  - 已发货（蓝色）
  - 已完成（绿色）
  - 已取消（灰色）

**订单发货**:
1. 找到状态为"已付款"的订单
2. 点击"发货"按钮
3. 输入快递单号
4. 点击"确定"
5. 订单状态自动变为"已发货"

#### 5. 客户管理
**查看用户**:
- 点击左侧菜单"客户管理"
- 查看所有注册用户
- 查看用户信息

**管理用户状态**:
- 找到需要管理的用户
- 点击状态开关
  - 开启：用户可以正常登录
  - 关闭：用户无法登录

#### 6. 浏览日志
**查看日志**:
- 点击左侧菜单"浏览日志"
- 查看所有浏览记录
- 查看用户浏览行为

**热门商品**:
- 查看浏览次数最多的商品
- 了解用户兴趣偏好

## 🔧 常见问题

### Q1: 如何启动项目？
**A**:
1. 启动后端：`./mvnw spring-boot:run`
2. 启动用户前台：`cd emart-web && npm run dev`
3. 启动管理后台：`cd emart-admin && npm run dev`

### Q2: 管理后台登录404错误？
**A**: 确保后端服务已启动，并重新编译加载了新的AdminController

### Q3: 购物车数据丢失？
**A**: 购物车数据存储在Redis中，默认30分钟过期。确保Redis服务正在运行。

### Q4: 图片无法显示？
**A**:
1. 确保图片已上传到 `D:\test\project\emart\uploads\` 目录
2. 检查后端静态资源映射配置
3. 确认图片路径格式正确

### Q5: 如何添加测试商品？
**A**:
1. 登录管理后台
2. 进入"商品管理"
3. 点击"添加商品"
4. 填写商品信息并提交

### Q6: 如何模拟下单流程？
**A**:
1. 注册用户并登录
2. 浏览商品，添加到购物车
3. 从购物车结算
4. 填写收货地址
5. 提交订单
6. 在管理后台发货

### Q7: 支付功能？
**A**: 目前支付功能尚未实现。点击"去支付"会提示"支付功能开发中"。

### Q8: 如何查看销售统计？
**A**:
1. 登录管理后台
2. 进入Dashboard首页
3. 查看统计数据卡片

## 🔐 安全说明

### 认证机制
- JWT Token认证
- Token有效期：604800秒（7天）
- Token存储：localStorage
- 自动携带Token请求

### 密码安全
- BCrypt加密存储
- 密码强度验证

### 路由权限
- 用户前台：未登录可浏览商品，需登录才能购物
- 管理后台：所有页面需要登录
