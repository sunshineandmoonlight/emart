## 🎨 商品图片配置

### 使用方案：在线占位图（placeholder.com）

所有商品图片已更新为在线占位图，特点：
- ✅ **无需本地文件**
- ✅ **加载速度快**
- ✅ **彩色区分不同商品**
- ✅ **显示商品名称文字**

### 图片URL示例：
```
https://via.placeholder.com/400x400/667eea/ffffff?text=iPhone+15+Pro+Max
```

**URL格式说明：**
- `400x400` - 图片尺寸
- `667eea` - 背景颜色（不同商品使用不同颜色）
- `ffffff` - 文字颜色（白色）
- `text=...` - 显示的文字（商品名称）

## 💡 后续优化建议

### 如果想要真实商品图片：

#### 方案1：替换为Unsplash图片
```sql
UPDATE pms_product
SET image = 'https://images.unsplash.com/photo-1592750475338-74b7b21085ab?w=400'
WHERE name LIKE '%iPhone%';
```

#### 方案2：使用淘宝/京东图片
1. 从电商平台下载商品图片
2. 保存到 `D:\test\project\emart\uploads\products\`
3. 更新数据库：
```sql
UPDATE pms_product SET image = '/uploads/products/iphone15.jpg' WHERE id = 13;
```

#### 方案3：保持占位图
- 当前方案已足够用于开发和测试
- 占位图清晰显示商品名称
- 无需额外配置，开箱即用
