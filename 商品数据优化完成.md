### 4. ✅ 商品编号系统

**格式：** SD + 分类ID(2位) + 商品ID(3位)

**示例：**
- iPhone 15 Pro Max → `SD01214` (分类1，商品214)
- 联想拯救者 → `SD02220` (分类2，商品220)
- 小米扫地机器人 → `SD03226` (分类3，商品226)

**生成规则：**
```sql
UPDATE pms_product
SET product_code = CONCAT('SD', LPAD(category_id, 2, '0'), LPAD(id, 3, '0'))
WHERE product_code IS NULL;
```

## 🔧 技术细节

### 数据库表结构
```sql
pms_product
├── id              BIGINT        主键
├── name            VARCHAR(200)  商品名称（中文）
├── subtitle        VARCHAR(500)  商品副标题（中文）
├── price           DECIMAL(10,2) 价格
├── stock           INT           库存
├── category_id     BIGINT        分类ID
├── brand           VARCHAR(200)  品牌
├── image           VARCHAR(500)  商品图片URL
├── status          INT           状态（1=上架）
├── product_code    VARCHAR(50)   商品编号
└── create_time     DATETIME      创建时间
```

### 商品编号规则
- **前缀：** SD (ShangDi - 商品)
- **分类码：** 2位数字，01-10
- **序号：** 3位数字，从001开始
- **示例：** SD01214 = SD + 分类01 + 商品214
